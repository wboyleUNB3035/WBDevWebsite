Group Java Application

First group project

This was my first group project, which a part of my second-year Software Engineering class. Our objective was to create a Java application that would analyze student transcripts and output various grade distributions.

This involved using many new technologies, Java properties, and software engineering principles that I had been previously unaware of. For example, Git, abstract classes, interfaces, SOLID principles, UML, and more. Our group was able to implement these properties and principles effectively to create a successful student transcript analyzer that could analyse Raw Distributions and Area Distributions (distributions averaged across different courses).

Reflecting on this experience, I learned how to effectively work in a group by communicating frequently, scheduling weekly standups, and compromise when needed. I'm happy that I was able to contribute significantly to a successful group project.
