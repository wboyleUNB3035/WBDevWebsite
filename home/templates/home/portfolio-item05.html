Unity Video Game

My first Game

This was the second project I've ever made, which was made over the course of a week during my quarantine upon moving to Fredericton.

Heavily inspired by the Brackey's video series, "How to Make a Video Game" I incorporated many of the concepts he used while also adding my own twists. I learned a lot about the Unity Game Engine, C#, and Game Design while making this project which I really enjoyed. I even added my own "Infinite" game mode which would create an endless version of the video game, similar to how many mobile games are designed nowadays.

I really enjoyed making this project because I've always wanted to learn how to make a video game but I never had the opportunity to. Additionally, I loved being able to design my own levels and I got to familiarize myself with one of the most popular game engines in the industry.
